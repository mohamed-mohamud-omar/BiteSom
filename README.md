# BiteSom
online food ordering
🎯 Concept

A system where users can view food items, add them to a cart, and place an order.

🧩 Core Features
1. 👤 User Authentication
Register (name, email, password)
Login / Logout
2. 🍔 View Food Menu
Display food items with:
Name
Price
Image
Categories (Burger, Pizza, Drinks)
3. 🛒 Add to Cart
Add food items to cart
Increase/decrease quantity
Remove item from cart
4. 💰 Checkout System
Show total price
Place order button
5. 📦 Order Management
Save orders in database
View previous orders
6. 🔍 Search & Filter
Search food by name
Filter by category
🛠️ Technologies
Frontend: React.js + TailwindCSS
Backend: Node.js + Express
Database: mongodb
